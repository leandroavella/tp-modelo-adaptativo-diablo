# TP: Modelo Adaptativo - Caso Diablo 2
**Alumno:** Leandro Avella

Este proyecto aplica el modelo de base de datos adaptativo para gestionar personajes de un RPG.
- Se configuró la DB con `utf8mb4` para soportar caracteres especiales.
- Se usaron **Opciones Únicas** para definir la Clase del personaje (Paladín, Bárbaro, etc.).
- Se usaron **Opciones Múltiples** para los Atributos (Fuerza, Vida) y Armas (Hachas, Varas).
